This repository contains R code to replicate our analysis of Scopus-indexed journals in ASEAN and East Asian countries (2015-2024).

Data not included - you must download it from SCImago.

## Download Data from SCImago

Go to https://www.scimagojr.com/journalrank.php and download 4 files:

### File 1: scimagojr_2015_all.csv
- Select year: 2015
- Click "Download journal list in this page"
- Save as: data/raw/scimagojr_2015_all.csv

### File 2: scimagojr_2015_oa.csv
- Keep year: 2015
- Under "Open Access" filter, check: Gold + Subsidized OA
- Click "Download journal list in this page"
- Save as: data/raw/scimagojr_2015_oa.csv

### File 3: scimagojr_2024_all.csv
- Select year: 2024
- Clear OA filter
- Click "Download journal list in this page"
- Save as: data/raw/scimagojr_2024_all.csv

### File 4: scimagojr_2024_oa.csv
- Keep year: 2024
- Under "Open Access" filter, check: Gold + Subsidized OA
- Click "Download journal list in this page"
- Save as: data/raw/scimagojr_2024_oa.csv
